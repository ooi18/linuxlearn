常用技巧
----
命令行快速跳转:
- ctrl + a 跳转到命令行开始位置
- ctrl + e 跳转到命令行结尾
- ctrl + u 删除光标到行首的字符
- ctrl + k 删除光标到行尾的字符
- ctrl + l 或者 clear 清理屏幕
- ctrl + w 清除前面一个单词
- Ctrl + r 显示：号提示，根据用户输入查找相关历史命令（reverse-i-search）
